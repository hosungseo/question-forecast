What it does

Question Forecast는 네이버 뉴스 API로 회의 전 이슈를 모으고, 과거 국무회의 대통령 발언 DB에서 관찰한 질문 패턴을 이용해 장관이 준비해야 할 질문 패킷을 생성합니다.
